# Weather-App
An AI-powered weather assistant built using LangChain and Groq that provides real-time weather information for any city using external weather APIs and LLM tool calling.
# Features
Get current weather details for any city
AI-powered responses using Groq LLM
Tool calling with LangChain agents

# Tech Stack
Python
LangChain
Groq API
Weather API
Uvicorn
