# Weather-App
An AI-powered weather assistant built using Streamlit, LangChain, Groq, and the OpenWeather API that provides real-time weather information for any city through a conversational interface.
# Features
Real-time weather information for any city
Conversational AI interface
LangChain agent with tool calling
Fast and efficient responses using Groq LLM
Interactive Streamlit web application
Weather details including:
Temperature
Humidity
Weather conditions
Wind speed

# Tech Stack
Python
Streamlit
LangChain
Groq API
OpenWeather API
